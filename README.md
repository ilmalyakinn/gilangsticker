## 📋 Deskripsi Proyek

Landing page ini dirancang untuk **Gilang Sticker & Branding**, sebuah bisnis spesialis branding kendaraan dan cutting sticker yang berlokasi di Cisewu, Garut, Jawa Barat. Website dibangun dengan tampilan premium bertema otomotif untuk meningkatkan kepercayaan pelanggan dan konversi bisnis.

Website ini sepenuhnya menggunakan **HTML, CSS, dan Vanilla JavaScript** murni tanpa framework atau library eksternal, sehingga ringan dan mudah di-deploy di mana saja.

---

## ✨ Fitur Utama

- **Fully Responsive** — Tampilan optimal di desktop, tablet, dan mobile
- **Sticky Navbar** — Transparan saat di atas, blur effect saat scroll
- **Hamburger Menu** — Navigasi mobile yang bersih
- **Scroll Reveal Animation** — Elemen muncul dengan animasi saat discroll
- **Portfolio Filter** — Filter karya berdasarkan kategori (Mobil, Motor, Kaca)
- **Smooth Scrolling** — Navigasi halaman yang mulus
- **Hover Effects** — Interaksi visual pada card dan tombol
- **WhatsApp CTA** — Tombol kontak langsung ke WhatsApp
